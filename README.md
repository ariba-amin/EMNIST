# Handwritten Character Recognition using CNN

## Project Overview

This project recognizes handwritten English alphabets using a Convolutional Neural Network (CNN). The model is trained on the EMNIST Letters dataset and can predict handwritten characters from images.

## Features

- EMNIST Letters Dataset
- Image Preprocessing
- CNN Model
- Model Training and Evaluation
- Handwritten Character Prediction

## Technologies Used

- Python
- TensorFlow
- TensorFlow Datasets
- NumPy
- Matplotlib
- Google Colab

## Model Performance

- Test Accuracy: **92%**

## Future Improvements

- Improve preprocessing for real-world handwritten images.
- Extend the model to recognize complete words and sentences using CRNN.
